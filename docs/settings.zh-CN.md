# 设置说明

点击界面右上角「设置」按钮打开设置面板。

## 📋 设置选项详情

| 设置项 | 默认值 | 说明 | 建议 |
| :--- | :--- | :--- | :--- |
| 📂 **下载目录** (downloadDir) | 未设置 | 拉取文件的默认保存位置 | 选择一个大容量、速度快的磁盘目录 |
| 🚀 **传输并发** (maxConcurrent) | 2 | 同时进行的最大传输任务数 (1-8) | **普通网络**：保持默认 2<br>**高速局域网**：可提高到 4-6<br>**网络不稳定**：降低到 1 |
| ⚡ **限速** (maxMbps) | 0（无限制） | 最大传输速度（Mbps） | 不想影响其他网络活动时设置限制。例如：100Mbps 网络可设为 50 |
| 🔍 **自动发现刷新间隔** (discoveryIntervalMs) | 5000（5秒） | 设备列表刷新间隔 | 保持默认，过短会增加网络负载 |
| 🌐 **仅显示同网段设备** (sameSubnetOnly) | 关闭 | 只显示同一子网内的设备 | **单一局域网**：保持关闭<br>**复杂网络环境**：开启可过滤无关设备 |
| 🔄 **更新镜像** (updateMirror) | https://ghfast.top/ | GitHub 访问加速镜像 | 中国大陆用户保持默认，海外用户可清空 |

## 🛠 如何修改设置

1. 点击右上角「设置」按钮
2. 修改需要的选项
3. 设置会自动保存
4. 部分设置需要重启生效

## 🧹 重置设置

- 点击「恢复默认」可重置所有设置
