# Settings Reference

Click the "Settings" button in the top right corner to open the settings panel.

## 📋 Settings Overview

| Setting | Default | Description | Recommendations |
| :--- | :--- | :--- | :--- |
| 📂 **Download Directory** (downloadDir) | Not set | Default location for pulled files | Choose a directory with plenty of space and fast disk I/O |
| 🚀 **Concurrent Transfers** (maxConcurrent) | 2 | Maximum number of simultaneous transfer tasks (1-8) | **Normal network**: Keep default 2<br>**High-speed LAN**: Increase to 4-6<br>**Unstable network**: Decrease to 1 |
| ⚡ **Speed Limit** (maxMbps) | 0 (unlimited) | Maximum transfer speed (Mbps) | Set a limit if you don't want transfers to use all bandwidth. Example: 50 for a 100Mbps network |
| 🔍 **Discovery Interval** (discoveryIntervalMs) | 5000 (5 seconds) | Device list refresh interval | Keep default; shorter intervals increase network load |
| 🌐 **Same Subnet Only** (sameSubnetOnly) | Off | Only show devices on the same subnet | **Single LAN**: Keep off<br>**Complex network**: Enable to filter irrelevant devices |
| 🔄 **Update Mirror** (updateMirror) | https://ghfast.top/ | GitHub access acceleration mirror | China users: keep default<br>International users: can leave empty |

## 🛠 How to Modify Settings

1. Click the "Settings" button in the top right
2. Modify the options you need
3. Settings are saved automatically
4. Some settings require an app restart to take effect

## 🧹 Reset Settings

- Click "Reset to Defaults" to restore all settings to their original values
